# Python トレーナー

ブラウザだけで動くPython学習ツールです。インストール不要。URLを開いてすぐにコードを書き始められます。

## 使い方

1. 下記のURLをブラウザで開く
2. 問題を選んでコードを書く
3. 「実行」ボタンを押して結果を確認する

**GitHub Pages URL**: https://bobotan02.github.io/python-trainer/

> ※ GitHub Pagesを有効化した後にアクセスできます。

## 特徴

- インストール・アカウント登録・サーバー不要
- ブラウザ上でPythonコードをそのまま実行（Pyodide使用）
- 初心者向けの問題を収録（beginner / intermediate / advanced）
- カテゴリ絞り込み・難易度フィルター付き

## 技術スタック

| 項目 | 内容 |
|------|------|
| 実行環境 | [Pyodide](https://pyodide.org/) 0.28.3（ブラウザ上でPythonを実行） |
| フロントエンド | HTML / CSS / JavaScript（フレームワークなし） |
| ホスティング | GitHub Pages |
| 外部依存 | Pyodide のみ（jsDelivr CDN経由） |
