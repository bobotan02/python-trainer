1. Pythonの基本構文（変数・型・演算子）

      
変数は「名前 = 値」で作ります。型宣言は不要で、値に応じて自動的に型が決まります。

      
name = "Taro"      # str（文字列）
age = 20           # int（整数）
height = 170.5     # float（小数）
is_student = True  # bool（真偽値: True / False）

print(type(age))   # <class 'int'>

      
主な演算子:

      
7 + 3    # 10  （和）
7 - 3    # 4   （差）
7 * 3    # 21  （積）
7 / 3    # 2.3333... （割り算。結果は必ずfloat）
7 // 3   # 2   （切り捨て割り算）
7 % 3    # 1   （余り）
7 ** 3   # 343 （べき乗）

# 比較演算子（結果は True / False）
a == b   # 等しい        a != b   # 等しくない
a < b    # より小さい    a >= b   # 以上

# 論理演算子
a and b  /  a or b  /  not a

      
⚠️ 初学者の躓きポイント: 「=」は代入、「==」は比較。if文で = と書くとエラーになります。

    


    

      
2. 制御構文（if / for / while）

      
Pythonはインデント（行頭の字下げ。半角スペース4つが標準）でブロックを表します。

      
# if / elif / else
score = 85
if score >= 80:
    print("合格")
elif score >= 60:
    print("再試験")
else:
    print("不合格")

# for（リストや range を順に処理）
for i in range(5):       # 0, 1, 2, 3, 4
    print(i)

for fruit in ["apple", "banana"]:
    print(fruit)

# while（条件が True の間繰り返す）
n = 3
while n > 0:
    print(n)
    n -= 1               # n = n - 1 と同じ

      
⚠️ range(5) は 0〜4 で、5 は含みません。range(1, 6) と書くと 1〜5 になります。break でループ中断、continue で次の周回へスキップできます。

    


    

      
3. 関数（def・return・引数）

      
def greet(name):
    return "こんにちは、" + name + "さん"

print(greet("Taro"))     # こんにちは、Taroさん

# デフォルト引数（呼び出し時に省略できる）
def power(base, exp=2):
    return base ** exp

print(power(5))          # 25（exp は省略 → 2）
print(power(5, 3))       # 125

# 複数の値を返す（実体はタプル）
def min_max(numbers):
    return min(numbers), max(numbers)

low, high = min_max([3, 1, 4, 1, 5])
print(low, high)         # 1 5

      
⚠️ return がない関数は None を返します。print(関数()) で None と表示されたら return を忘れていないか確認しましょう。

    


    

      
4. データ構造（list・dict・tuple・set）

      
# list（リスト）— 順序あり・変更可
nums = [10, 20, 30]
nums.append(40)          # 末尾に追加 → [10, 20, 30, 40]
nums[0]                  # 10（先頭。インデックスは0から）
nums[-1]                 # 40（末尾）
len(nums)                # 4（要素数）

# dict（辞書）— キーと値のペア
user = {"name": "Taro", "age": 20}
user["name"]             # "Taro"
user["city"] = "Tokyo"   # 追加
user.keys()              # キー一覧
user.values()            # 値一覧
user.items()             # (キー, 値) のペア一覧

# tuple（タプル）— 順序あり・変更不可
point = (3, 5)
x, y = point             # アンパック代入

# set（集合）— 重複なし・順序なし
s = {1, 2, 2, 3}         # → {1, 2, 3}（重複は自動で消える）
1 in s                   # True（高速な所属判定）

      
⚠️ リストのインデックスは0始まり。存在しない辞書キーへのアクセスは KeyError になります（user.get("x") なら None が返り安全）。

    


    

      
5. 内包表記

      
リストなどを1行で生成する書き方です。「for文＋append」を簡潔に書けます。

      
# 基本形: [式 for 変数 in イテラブル]
squares = [x ** 2 for x in range(5)]
# → [0, 1, 4, 9, 16]

# 条件付き: [式 for 変数 in イテラブル if 条件]
evens = [x for x in range(10) if x % 2 == 0]
# → [0, 2, 4, 6, 8]

# 文字列にも使える
upper = [s.upper() for s in ["a", "b", "c"]]
# → ['A', 'B', 'C']

# 辞書内包表記
d = {x: x * 10 for x in range(3)}
# → {0: 0, 1: 10, 2: 20}

      
⚠️ 「式」が先、「for」が後という語順に注意。慣れるまでは同じ処理をfor文で書いてから変換すると理解しやすいです。

    


    

      
6. クラスの基本

      
class Dog:
    def __init__(self, name):    # コンストラクタ
        self.name = name         # インスタンス変数

    def bark(self):              # メソッド
        return self.name + ": ワン！"

pochi = Dog("ポチ")              # インスタンス生成
print(pochi.bark())              # ポチ: ワン！

# 継承（親クラスの機能を引き継ぐ）
class PoliceDog(Dog):
    def bark(self):              # メソッドの上書き（オーバーライド）
        return self.name + ": ワンワン！！"

k9 = PoliceDog("ケン")
print(k9.bark())                 # ケン: ワンワン！！

      
⚠️ メソッドの第1引数には必ず self を書きます（呼び出し時には渡しません）。self を忘れると TypeError になります。親クラスのメソッドは super().メソッド名() で呼べます。

    


    

      
7. 例外処理（try・except）

      
try:
    n = int("abc")           # ここで ValueError が発生
    print(n)                 # ← 実行されない
except ValueError:
    print("数値に変換できません")

# 複数の例外・エラー内容の取得
try:
    result = 10 / 0
except ZeroDivisionError as e:
    print("エラー:", e)      # エラー: division by zero
finally:
    print("必ず実行される")   # 成否にかかわらず実行

# 自分で例外を発生させる
def set_age(age):
    if age < 0:
        raise ValueError("年齢は0以上にしてください")

      
⚠️ except を書きすぎて本来直すべきバグまで握りつぶさないように。捕まえる例外の種類（ValueError など）は具体的に指定するのが基本です。

print(値)          # 画面に出力する
len(x)             # 要素数・文字数        len("abc") → 3
type(x)            # 型を調べる            type(1.5) → <class 'float'>
int(x) / float(x) / str(x)   # 型変換      int("42") → 42
abs(x)             # 絶対値                abs(-5) → 5
round(x, n)        # 四捨五入（n桁）       round(3.14159, 2) → 3.14
min(...) / max(...) # 最小・最大           max([3, 1, 4]) → 4

enumerate(リスト)  # 番号付きで取り出す    for i, v in enumerate(...)
zip(a, b)          # 2つの列を同時に処理   for x, y in zip(a, b)
input()            # ※本ツールでは使用不可（標準入力非対応）

      
⚠️ sorted() は新しいリストを返し、リスト.sort() は元のリスト自体を並べ替えて None を返します。print(nums.sort()) が None になるのは典型的な躓きポイントです。

    


    

      
9. よく使うstring操作

      
s = "Hello, Python"

# インデックスとスライス
s[0]        # "H"（1文字目）
s[-1]       # "n"（末尾）
s[0:5]      # "Hello"（0番目から5番目の手前まで）
s[7:]       # "Python"（7番目から最後まで）
s[::-1]     # 逆順（"nohtyP ,olleH"）

# 大文字・小文字
s.upper()   # "HELLO, PYTHON"
s.lower()   # "hello, python"

# 置換・分割・結合
s.replace("Python", "World")   # "Hello, World"
"a,b,c".split(",")             # ['a', 'b', 'c']
"-".join(["a", "b", "c"])      # "a-b-c"

# 検索・判定
s.find("Py")        # 7（見つからなければ -1）
"Py" in s           # True
s.startswith("He")  # True
"  abc  ".strip()   # "abc"（前後の空白を除去）

# f文字列（変数の埋め込み）
name, age = "Taro", 20
f"{name}は{age}歳"             # "Taroは20歳"
f"{3.14159:.2f}"               # "3.14"（小数2桁）
"{}は{}歳".format(name, age)   # format方式でも同じ

      
⚠️ 文字列は変更不可（イミュータブル）。s.upper() は新しい文字列を返すだけで、s 自体は変わりません。結果を使うには s = s.upper() のように代入が必要です。

    


    

      
10. 記法の読み方（このあとのセクションを読む前に）

      
これから登場するSQL・正規表現などのコード例には、これまでとは少し違う見た目の記法が出てきます。先に4パターンを押さえておくと、各セクションのコードが読みやすくなります。

      
    
        
  • モジュール名.関数名() — 例: re.search(...)、json.dumps(...)。import reのように読み込んだ「モジュール」（機能をひとまとめにした部品）の中にある関数を呼び出しています。まず「どのモジュールの関数か」に注目すると、何をするコードか推測しやすくなります。
    • 
        
  • 変数名.メソッド名() — 例: cur.execute(...)、"abc".strip()。こちらは変数（オブジェクト）自身が持っている機能を呼び出す書き方で、6節のクラスで出てきた「メソッド」と同じ仕組みです。見た目は「モジュール名.関数名()」とよく似ていますが、ドットの左側が「モジュール名」なのか「変数名」なのかで意味が変わるので注意してください。
    • 
        
  • 先頭が大文字のClassName(...) — 例: datetime(2024, 1, 1)、Counter([...])。Pythonにはクラス名を先頭大文字で書く慣習（PascalCaseと呼びます）があり、この形は多くの場合「そのクラスから新しいオブジェクトを作っている」ことを表します。
    • 
        
  • 文字列の中の大文字の並び — 例: "SELECT * FROM t"のSELECTなど。これはPythonの文法ではありません。ただの文字列としてそのままsqlite3のような別のライブラリに渡され、SQLという別の言語として解釈される部分です。Python自身はこの中身を解釈しないので、SQLの綴りを間違えてもPython側ではエラーになりません（実行してみて初めてSQL側のエラーとして出ます）。
    • 
      

      
import re
from datetime import datetime

print(re.search(r"\d+", "abc123").group())   # 123（モジュール re の関数）
print("  abc  ".strip())                      # abc（変数=文字列 が持つメソッド）
print(datetime(2024, 1, 1))                   # 2024-01-01 00:00:00（クラスからオブジェクトを作成）

sql = "SELECT * FROM t"
print(sql)                                     # SELECT * FROM t（ただの文字列。SQLとして解釈されるのは実行時）

      
⚠️ 迷ったら「ドットの左は変数か、モジュールか」「先頭が大文字か」「文字列の中かどうか」の3点を確認すると、初めて見る記法でもおおよその意味が推測できます。

    


    

      
11. SQL入門（sqlite3）

      
Pythonの標準ライブラリ sqlite3 を使うと、追加インストールなしでSQLデータベースを扱えます。:memory: を指定するとメモリ上に一時的なデータベースを作れます。

      
import sqlite3

conn = sqlite3.connect(":memory:")
cur = conn.cursor()

# テーブル作成とデータ挿入
cur.execute("CREATE TABLE users (name TEXT, age INTEGER, city TEXT)")
cur.executemany("INSERT INTO users VALUES (?, ?, ?)", [
    ("Taro", 20, "Tokyo"), ("Hanako", 25, "Osaka"), ("Jiro", 32, "Tokyo"),
])
conn.commit()

# SELECT / WHERE / ORDER BY
for row in cur.execute("SELECT name, age FROM users WHERE age >= 25 ORDER BY age DESC"):
    print(row)
# ('Jiro', 32)
# ('Hanako', 25)

# GROUP BY / HAVING（都市ごとに人数と平均年齢）
for row in cur.execute("SELECT city, COUNT(*), AVG(age) FROM users GROUP BY city HAVING COUNT(*) >= 1"):
    print(row)
# ('Osaka', 1, 25.0)
# ('Tokyo', 2, 26.0)

# JOIN（2つのテーブルを name で結合）
cur.execute("CREATE TABLE scores (name TEXT, score INTEGER)")
cur.executemany("INSERT INTO scores VALUES (?, ?)", [("Taro", 80), ("Jiro", 90)])
conn.commit()
for row in cur.execute("SELECT users.name, scores.score FROM users JOIN scores ON users.name = scores.name"):
    print(row)
# ('Taro', 80)
# ('Jiro', 90)

# プレースホルダ（?）— 値を安全に埋め込む
cur.execute("SELECT name FROM users WHERE city = ?", ("Tokyo",))
print(cur.fetchall())   # [('Taro',), ('Jiro',)]

      
⚠️ ORDER BY を書かないと行の順序は保証されません（たまたま挿入順に見えても仕様ではない）。execute() の1行はタプルで返ってきます。値をSQL文に文字列結合で埋め込むと壊れたりSQLインジェクションの原因になるため、必ずプレースホルダ（?）を使ってください。

    


    

      
12. 正規表現（re）

      
文字列がある「パターン」に合っているかを調べたり、パターンに合う部分を取り出したりするための標準ライブラリです。パターンには \d（数字）のような特殊な記号を使うため、書くときは r"..."（raw文字列）にするのが基本です。

      
import re

# search / match / fullmatch の違い
print(re.search(r"\d+", "abc123xyz").group())     # 123（文字列のどこかにあればOK）
print(re.match(r"\d+", "abc123xyz"))              # None（先頭からじゃないとダメ）
print(re.match(r"[a-z]+", "abc123xyz").group())   # abc（先頭からマッチ）
print(re.fullmatch(r"[a-z]+\d+", "abc123"))        # マッチ（文字列全体が対象）
print(re.fullmatch(r"[a-z]+\d+", "abc123xyz"))     # None（末尾のxyzが余る）

# findall — マッチした部分を全部リストで取得
print(re.findall(r"\d+", "a1b22c333"))    # ['1', '22', '333']

# sub — マッチした部分を置換
print(re.sub(r"\d+", "#", "a1b22c333"))   # a#b#c#

# グループ（）で部分を取り出す
m = re.search(r"(\d{4})/(\d{2})/(\d{2})", "2026/07/01")
print(m.group(1), m.group(2), m.group(3))          # 2026 07 01

# 名前付きグループ（?P<名前>...）
m2 = re.search(r"(?P<year>\d{4})/(?P<month>\d{2})/(?P<day>\d{2})", "2026/07/01")
print(m2.group("year"), m2.group("month"))          # 2026 07

# split — パターンで分割
print(re.split(r"[,;]\s*", "a, b; c,d"))   # ['a', 'b', 'c', 'd']

      
よく使うメタ文字: \d数字 / \w英数字とアンダースコア / \s空白 / +1回以上 / *0回以上 / ?0か1回 / {n}ちょうどn回 / []文字の集合 / ^先頭 / $末尾

      
⚠️ match() は文字列の先頭からしかマッチを見ません（途中にあるものは search() を使う）。.（ドット）は改行以外のほぼ何にでもマッチするので、ピリオドそのものを表したいときは \. のようにエスケープが必要です。

    


    

      
13. 日付と時刻（datetime）

      
from datetime import datetime, date, timedelta

# strptime — 文字列 → datetime（parseのp）
d = datetime.strptime("2026-07-01", "%Y-%m-%d")
print(d)             # 2026-07-01 00:00:00
print(type(d))       # <class 'datetime.datetime'>

# strftime — datetime → 文字列（formatのf）
print(d.strftime("%Y年%m月%d日"))   # 2026年07月01日

# timedelta — 日数・時間を足し引きする
d2 = d + timedelta(days=10)
print(d2)            # 2026-07-11 00:00:00

# 日付の差・比較
d3 = datetime.strptime("2026-07-15", "%Y-%m-%d")
print((d3 - d).days) # 14
print(d3 > d)        # True

print(date(2026, 7, 1))   # 2026-07-01（時刻を持たない日付だけの型）

      
主な書式指定子: %Y西暦4桁 / %m月(01-12) / %d日(01-31) / %H時(00-23) / %M分 / %S秒

      
⚠️ strptime（p=parse、文字列→datetime）と strftime（f=format、datetime→文字列）は矢印の向きが逆です。名前が似ているので混同しやすい躓きポイントです。

    


    

      
14. collections・itertools

      
from collections import Counter, defaultdict, namedtuple
import itertools

# Counter — 出現回数を数える
c = Counter(["a", "b", "a", "c", "a", "b"])
print(c)                  # Counter({'a': 3, 'b': 2, 'c': 1})
print(c.most_common(2))   # [('a', 3), ('b', 2)]（多い順に上位n件）

# defaultdict — キーが無いときの初期値を自動で用意する
dd = defaultdict(list)
dd["fruits"].append("apple")
dd["fruits"].append("banana")
print(dict(dd))   # {'fruits': ['apple', 'banana']}

# namedtuple — フィールド名でアクセスできるタプル
Point = namedtuple("Point", ["x", "y"])
p = Point(1, 2)
print(p.x, p.y)    # 1 2
print(p)           # Point(x=1, y=2)

# itertools.groupby — 同じキーが連続している部分だけをまとめる
data = [("Tokyo", 1), ("Osaka", 2), ("Tokyo", 3)]
data_sorted = sorted(data, key=lambda x: x[0])   # 先にキーでソートしておく
for key, group in itertools.groupby(data_sorted, key=lambda x: x[0]):
    print(key, list(group))
# Osaka [('Osaka', 2)]
# Tokyo [('Tokyo', 1), ('Tokyo', 3)]

# itertools.chain — 複数のリストをつなげて1つとして扱う
print(list(itertools.chain([1, 2], [3, 4])))   # [1, 2, 3, 4]

      
⚠️ itertools.groupby は「隣り合っている同じキー」しかまとめません。事前にソートしていないと、同じキーでも離れた場所にあると別グループとして扱われてしまいます（上の例でソートを省くと Tokyo が2グループに分かれます）。

    


    

      
15. JSON・CSVの読み書き

      
import json, csv, io

# json.dumps / loads
data = {"name": "太郎", "age": 20}
print(json.dumps(data))                              # {"name": "太郎", "age": 20}
print(json.dumps(data, ensure_ascii=False))           # {"name": "太郎", "age": 20}
print(json.dumps(data, ensure_ascii=False, indent=2))
# {
#   "name": "太郎",
#   "age": 20
# }
print(json.loads('{"name": "太郎", "age": 20}'))       # {'name': '太郎', 'age': 20}

# csv.reader — 1行ずつリストで取得
csv_text = "name,age\nTaro,20\nHanako,25"
for row in csv.reader(io.StringIO(csv_text)):
    print(row)
# ['name', 'age']
# ['Taro', '20']
# ['Hanako', '25']

# csv.DictReader — 1行ずつ辞書（列名がキー）で取得
for row in csv.DictReader(io.StringIO(csv_text)):
    print(row, type(row["age"]))
# {'name': 'Taro', 'age': '20'} <class 'str'>
# {'name': 'Hanako', 'age': '25'} <class 'str'>

      
⚠️ json.dumps() はデフォルトで日本語を \uXXXX にエスケープします（ensure_ascii=False で回避）。CSVの値はヘッダーも中身もすべて文字列として読み込まれるので、数値として使うには int() / float() で変換が必要です。

    


    

      
16. pandas入門（参考・読み物）

      
⚠️ pandasは本ツールでも実行できます（import pandasを含むコードを実行すると自動で読み込まれます。初回のみ数秒かかります。2回目以降はブラウザにキャッシュされ高速になります）。ただしDataFrameの表示（print(df)など）はpandasのバージョンによって列幅の空白が変わることがあるため、このツールの問題の正誤判定には使われていません。

      
pandasとは: 表形式データ（CSV・Excel・データベースの表など）を読み込み・加工・集計するためのデータ分析ライブラリです。データの前処理・集計・分析の定番ツールで、機械学習の前段階のデータ整形にも広く使われます。慣例として「import pandas as pd」と略称で読み込みます。

      
主要な3つの概念:

      
    
        
  • DataFrame — 行と列を持つ2次元の表。Excelのシートのようなイメージで、pandasの中心となるデータ構造です。
    • 
        
  • Series — 1列分のデータ（ラベル付きの1次元配列）。DataFrameから1列取り出すとSeriesになります。
    • 
        
  • Index — 各行・各列を識別するラベル。行の見出し（既定では 0, 1, 2, ...）と列名がこれにあたります。
    • 
      

      
import pandas as pd

# Series — 1列分のデータ（ラベル付き）
s = pd.Series([10, 20, 30], index=["a", "b", "c"])
s["b"]                  # 20（ラベルで値を取り出せる）

# DataFrame — 2次元の表（辞書から作成）
df = pd.DataFrame({
    "name": ["Taro", "Hanako", "Jiro"],
    "age":  [20, 25, 32],
    "city": ["Tokyo", "Osaka", "Tokyo"],
})
#      name  age   city
# 0    Taro   20  Tokyo
# 1  Hanako   25  Osaka
# 2    Jiro   32  Tokyo

df.index                # 行ラベル: RangeIndex(start=0, stop=3, step=1)
df.columns              # 列名:     Index(['name', 'age', 'city'], ...)

      
よく使う操作:

      
    
        
  • read_csv — CSVファイルを読み込んでDataFrameを作る（入口になる操作）
    • 
        
  • head / 列選択 / 条件抽出 — データの中身を確認し、必要な部分を取り出す
    • 
        
  • groupby — 「〜ごとに集計」（例: 都市ごとの平均年齢）
    • 
        
  • merge — 共通の列をキーに2つの表を結合する（SQLのJOINに相当）
    • 
        
  • fillna — 欠損値（NaN）を指定した値で埋める
    • 
        
  • sort_values — 指定した列の値で並べ替える
    • 
      

      
# CSVの読み込み（ファイルの1行目が列名になる）
df = pd.read_csv("data.csv")
df.head()               # 先頭5行を表示（データ確認の第一歩）

# 列の取り出しと条件抽出
df["age"]               # 1列を取り出す（結果はSeries）
df[["name", "age"]]     # 複数列を取り出す（結果はDataFrame）
df[df["age"] >= 25]     # 条件を満たす行だけ抽出
#      name  age   city
# 1  Hanako   25  Osaka
# 2    Jiro   32  Tokyo

# groupby — 都市ごとの平均年齢
df.groupby("city")["age"].mean()
# city
# Osaka    25.0
# Tokyo    26.0

# merge — name をキーに2つの表を結合
scores = pd.DataFrame({"name": ["Taro", "Jiro"], "score": [80, 90]})
pd.merge(df, scores, on="name", how="left")
#      name  age   city  score
# 0    Taro   20  Tokyo   80.0
# 1  Hanako   25  Osaka    NaN   ← 対応がない行は NaN（欠損値）
# 2    Jiro   32  Tokyo   90.0

# fillna — 欠損値（NaN）を 0 で埋める
data = pd.DataFrame({"sales": [100, None, 300]})
data.fillna(0)
#    sales
# 0  100.0
# 1    0.0
# 2  300.0

# sort_values — age の大きい順に並べ替え
df.sort_values("age", ascending=False)

      
⚠️ 再掲: 上記のコードは問題タブ・ステップ実行タブのエディタに貼り付けて実行できます（初回実行時はpandasの読み込みで数秒かかります）。ただしdf.groupby(...)のように最後の式だけを書く行は、そのままでは画面に何も表示されません（本ツールはprint()した内容だけを出力欄に表示します）。試す場合はprint(df.groupby("city")["age"].mean())のようにprint()で囲んでください。
